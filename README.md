# @dzo/com

集公共组件库开发、调试、编写文档一体化

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```

发包

```bash
$ npm run release
```
